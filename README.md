## IoT 스마트 화분
### IotFlowerpot

ESP8266 + AWS + Android

## Function
  1. 공기 온도, 습도, 토양 수분 모니터링
  2. ESP8266 Web Server를 이용한 Wi-Fi 초기 설정
  3. Android App을 이용한 물 공급
  4. 토양 수분을 감지하여 자동으로 물 

## Hardware
  1. NodeMCU ESP8266
  2. DHT11
  3. 워터펌프
  4. 토양 습도센서

## Tools
  1. Python Flask
  2. AWS(DynamoDB, IAM, IoT Core, EC2)
  3. Android Studio
  4. Arduino
  5. MQTT
